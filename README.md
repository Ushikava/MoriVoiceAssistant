# Voice Assistant
Voice assistant based by STT, TTS, and GPT-technology.
Project on Python.
GUI on HTML/CSS.

## Description
This application is a voice assistant, an interlocutor, based on the information provided to him by the user. Features that distinguish the development can be noted the algorithm of arbitrary text generation, as well as the mascot. The GPT algorithm is trained on messages from the Discord channel. The sample size is made up of data for five years.
